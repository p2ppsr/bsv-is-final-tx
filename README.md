# bsv-is-final-tx

Checks if a BSV transaction is final

## Installation

```
npm i bsv-is-final-tx
```

## Usage

```js
const isFinal = require('bsv-is-final-tx')

(async () => {
  console.log(await isFinal('YOUR_TX_HEX_HERE'))
})()
```
